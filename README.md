#죽음의 스쿼트배틀 : PUMPETITION

### 환영의 말
환영합니다.
본 프로젝트는 Pose-Estimation 을 기반으로 한 동작인식 헬스게임 앱입니다.

2019 OpenHack Contest 출품작입니다.
